# Soul & Identity

## The Project's Soul
We believe in absolute transparency when it comes to AI usage. This project exists to empower developers with real-time awareness of their agentic overhead and provider limits.

## Guiding Principles
1. **Real-time Precision**: Quotas are only useful if they are accurate.
2. **Minimal Friction**: The TUI should be fast, responsive, and out of the way.
3. **Cross-Agent Visibility**: A single dashboard for all agents in the FMG ecosystem.
